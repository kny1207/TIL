  1. 연락처 입력

  3. 연락처 삭제

* 연락처 입력시
  * 이름: 2글자 이상 4글자 이하로 조건 줘서 범위 벗어나지 않도록
  * 전화번호: 형식이 맞아야, 앞에 3자리 뒤에 4자리 뒤에 4자리로 고정해야

* 연락처 삭제
